Transaction History

This section displays the history of all transactions made by users. You can view transaction details including timestamps, values, and status (fraudulent or legitimate).

  • Transaction ID: Unique identifier for each transaction.
  • User ID: Identifier for the user associated with the transaction.
  • Purchase Time: The timestamp of when the transaction was made.
  • Purchase Value: The monetary value of the transaction.
  • Status: Indicates whether the transaction is fraudulent or legitimate.
