Dataset Overview

This dataset contains transaction data used for fraud detection, comprising the following attributes:

  • user_id: Unique identifier for each user.
  • signup_time: Timestamp indicating when the user signed up.
  • purchase_time: Timestamp for when a purchase was made.
  • purchase_value: The monetary value of the purchase.
  • device_id: Identifier for the device used for the transaction.
  • source: The origin of the traffic (e.g., organic, paid).
  • browser: The web browser used for making the transaction.
  • sex: Gender of the user.
  • age: Age of the user.
  • ip_address: The IP address from which the user accessed the service.
  • country: The country of the user, indicating the geographical location.
  • class: Indicates whether the transaction is fraudulent or legitimate.
